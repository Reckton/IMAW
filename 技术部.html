github

【前言】 以下作业仅为提高性作业，不要求每个人必须完成。作业中涉及git的部分全部要求通过git命令行完成。
ps:"D盘"仅作为参考，可以选择任意盘符。
【具体作业】
1.  clone一个仓库到本地的"D:\IMAW\git"文件目录下。
2.  在"D:\IMAW\git\MyFirstRepository"文件目录下新建任意文本文件，内容任意。将"D:\IMAW\git\MyFirstRepository"下的全部内容，提交到远程仓库（github）。
3. 修改文件目录下的任意文件，并提交修改到github。
4. 通过git命令行查看目录下，发生过改变的内容
。 5.  将文件目录下的内容，回滚到最初始的版本。
【提交说明】
提交word文件，命名为"信协-技术部-第一次培训-18级XXX（姓名）"。
word内，按先后顺序放置关键信息的截图（主要包括命令行的命令、文件夹目录情况、相关操作执行成功结果）。 此外，也可以在结尾处，整理习得的知识点和对git的理解，以及出现的问题和解决的思路。这样能更好的成长。
【注】
以后所有的作业统一提交到IMAW Tech R群各自名下的文件夹内。
